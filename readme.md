# Skill Search

Skill Search is a simple **Django-based networking platform** that enables users to connect based on skills. Users can search for others by skills, send connection requests, accept/reject them, and message accepted connections. It's designed with a focus on backend functionality, using basic HTML/CSS for the frontend.

## Features
1. *User Authentication*: Built-in login, registration, and logout.
2. *Profile Management*: Edit skills (comma-separated) in your profile.
3. *Skill Search*: Search users by skills and send connection requests.
4. *Connections*: View pending requests, accept/reject, and list accepted connections.
5. *Messaging*: Send and view messages with accepted connections.

## Technologies
+ *Backend*: Django 5.0
+ *Database*: SQLite (default; configurable)
+ *Environment*: Python 3.11+
